# Success Criteria

## User Success

- 학습한 단어 중 **Cognitive Level 4 이상(생산형 단계) 비율이 50% 초과** 시 실력 향상으로 인식
- **일일 세션 완주율 80% 이상** 유지 (학습 습관 형성 지표)
- 동일 단어 재오답률이 **2주 내 50% 감소** (SRS 효과 검증)
- "오늘 복습할 단어가 있다"는 알림이 부담이 아닌 기대감으로 느껴지는 상태

## Business Success

- **3개월 내** 학습 단어 500개 이상 등록 및 Level 분포 데이터 축적
- 향후 멀티유저 확장 시 온보딩 없이 바로 사용 가능한 UX 완성도

## Measurable Outcomes

| 지표 | MVP 목표 | 측정 방법 |
|---|---|---|
| Level 4+ 단어 비율 | ≥ 50% | DB 집계 |
| 평균 레이턴시 | < 2초 | 클라이언트 타이머 |
| 일일 완주율 | ≥ 80% | 세션 로그 |
| 재오답률 2주 감소 | ≥ 50% | 오답 이력 비교 |
