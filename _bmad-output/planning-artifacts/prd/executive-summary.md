# Executive Summary

VOCA는 TOEFL 고난도 어휘 학습자가 겪는 **인지적 병목(Context Gap)** 을 해소하는 개인 맞춤형 독해 유창성 훈련 시스템이다. 기존 플래시카드·단어장 앱이 측정하지 못한 **인지 처리 속도(반응 레이턴시)** 를 핵심 숙련도 지표로 삼아, 단어 암기를 넘어 "지문의 논리 구조가 투명하게 보이는 Fluency" 달성을 목표로 한다.

**타겟 사용자:** TOEFL Reading 고득점을 목표로 하는 학습자 (초기: 개인 사용자, 향후 확장)

**해결 문제:** precipitously를 '급격하게'라고 암기했음에도, 실제 지문 독해 시 [단어 인식 → 한국어 변환 → 문맥 결합] 3단계 인지 과정이 발생해 독해 흐름이 끊기는 현상. 단어를 고립된 기호로 외워 맥락·이미지·논리 기능이 내재화되지 않은 것이 근본 원인이다.

**성공 기준:** TOEFL Reading 지문에서 어휘로 인한 Regression(역행 독해) 빈도가 현저히 감소하고, 원인-결과·대조 등 논리 구조가 즉각적으로 파악되는 상태.

## What Makes This Special

| 기존 앱 | VOCA |
|---|---|
| "정답/오답" 이분법 | 반응 레이턴시(ms) 기반 숙련도 측정 |
| 단어 = 암기 대상 | 단어 = 인지 훈련 데이터 포인트 |
| 고정된 3종 퀴즈 | Cognitive Level(1~5)에 따른 Adaptive Quiz Engine |
| 사전 예문 | 실제 TOEFL 코퍼스 기반 Collocation + 논리 역할 태깅 |

**핵심 인사이트:** "단어를 안다"는 것은 이분법이 아닌 5단계 연속 스펙트럼(인식→회상→이해→생산→자동화)이며, 레이턴시 데이터가 각 단어의 실제 숙련 단계를 객관적으로 결정한다.
